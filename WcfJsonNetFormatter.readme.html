Welcome to WcfJsonNetFormatter

WcfJsonNetFormatter is a library which implements WcfJsonNetFormatter component in order to transform message body into Json format, using the powerful Json.Net serializer.

Latest Version

The quickest way to get the latest release of WcfJsonNetFormatter is to add it to your project using NuGet (http://nuget.org/List/Packages/WcfJsonNetFormatter).
